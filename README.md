## react-weather-app
> **OpenWeatherMap API를 이용한 날씨 앱**

## 프로젝트 소개
현재위치 기반의 날씨뿐 만 아니라, 5개의 버튼(현재위치, 4개의 다른도시)를 클릭하여 도시별 날짜를 볼 수 있다.
데이터를 들고 오는 동안 로딩 스패너가 작동되며, 앱이 실행되는 첫 시점에는 현재 위치를 가져와 날씨를 보여준다.

## 시작 가이드
### Installation
``` bash
$ cd react-weather-app
$ npm install
$ npm start
```
